\chapter{Conventions}

\section*{Typefaces}

Latin\-/script Interturkic text is set in a serif typeface,
e.g.\ \latntxt{baş}, \latntxt{fikr}, \latntxt{tığ}.
Arabic\-/script Interturkic text is set in a Naskh\-/like typeface,
e.g.\ \arabtxt{باش}, \arabtxt{فکر}, \arabtxt{تیغ}.
When Latin\-/script text from any other language\---including Interturkic\---is
embedded within English\-/language prose, it is italicised,
e.g.\ \foreign{\latntxt{baş}}, \foreign{\latntxt{fikr}}, \foreign{\latntxt
{tığ}}.
Glossing abbreviations are set in small caps,
e.g.\ \Fpl, \Abl, \Pst.

\section*{Transcriptions}

All phonological transcriptions are rendered with symbols from the International
Phonetic Alphabet~(IPA).

Variable segments are represented by uppercase letters.
`A' represents the set \phonem{ɑ, ɛ}.
`D' represents the set \phonem{d, t}.
`G' represents the set \phonem{g, ɣ, k, q}.
`I' represents the set \phonem{ɯ, i, u, y}.
`K' represents the set \phonem{k, q}.
This is the case in both phonological transcriptions and in Latin\-/script
Interturkic text,
e.g.\ \foreign{\latntxt{\morphophon{\-/nIñ}}} \phonem{\morphophon{\-/nIŋ}}
\inlinegls{\Gen}.

\section*{Punctuation}

\subsection*{Phonology}

Phonemic transcriptions display underlying phonemes whose differences from
each other meaningfully contribute to an utterance's meaning.
They are enclosed within forward slashes,
e.g.\ \phonem{ɑwt͡ʃɯ}, \phonem{ijnɛ}, \phonem{ɑŋlɑmɑq}.
Phonetic transcriptions display actually realised sounds in speech,
representing the results of any sound changes that do not fundamentally alter
an utterance's meaning.
They are enclosed within square brackets,
e.g.\ \phonet{ɑwd͡ʒɯ}, \phonet{iːnɛ}, \phonet{ɑŋnɑmɑq}.

\subsection*{Morphophonology}

In both phonological transcriptions and in Latin\-/script Interturkic text,
segments that vary according to morphophonological factors are enclosed within
curly brackets,
e.g.\ \foreign{\latntxt{\morphophon{\=/sA}}} \phonem{\morphophon{\=/sA}}
\inlinegls{\Cond}.

At the beginning of suffixes, parentheses enclose conditionally included
segments.
An enclosed vowel appears only if preceded by a consonant,
e.g.\ \foreign{\latntxt{üy}} +~\foreign{\latntxt{\morphophon{\=/(I)ñ}}}
\derivesto~\foreign{\latntxt{üy\=/üñ}},
but \foreign{\latntxt{bala}} +~\foreign{\latntxt{\morphophon{\=/(I)ñ}}}
\derivesto~\foreign{\latntxt{bala\=/ñ}}
An enclosed consonant appears only if preceded by a vowel,
e.g.\ \foreign{\latntxt{altı}} +~\foreign{\latntxt{\morphophon{\=/(ş)Ar}}}
\derivesto~\foreign{\latntxt{altı\=/şar}},
but \foreign{\latntxt{bir}} +~\foreign{\latntxt{\morphophon{\=/(ş)Ar}}}
\derivesto~\foreign{\latntxt{bir\=/er}}.
Anywhere else in a suffix, parentheses enclose optional segments,
e.g.\ \foreign{\latntxt{öz}} +~\foreign{\latntxt{\morphophon{\=/(s)I}}}
+~\foreign{\latntxt{\morphophon{\=/n(I)}}} \derivesto~\foreign{\latntxt{öz\=/ü%
\=/nü}}/\foreign{\latntxt{öz\=/ü\=/n}}.

\subsection*{Morphology}

Dashes may be used to separate morphemes in IPA transcriptions, glosses, and
Latin\-/script text.
A dash to the right of an element marks it as a verbal stem,
e.g.\ \foreign{\latntxt{al\=/}} \transl{take}, \foreign{\latntxt{ket\=/}}
\transl{go}, \foreign{\latntxt{tur\=/}} \transl{stand}.
A dash to the left of an element marks it as a suffix,
e.g.\ \foreign{\latntxt{\morphophon{\=/çI}}} \inlinegls{\Denom}, \foreign{%
\latntxt{\morphophon{\=/GAn}}} \inlinegls{\Ptcp.\Pst}, \foreign{\latntxt{%
\morphophon{\=/lIK}}} \inlinegls{\Denom}.

Conversely, full stops join glossing elements together when they constitute a
single morpheme,
e.g.\ \foreign{\latntxt{\morphophon{\=/(s)I}}} \inlinegls{\Poss.\Third},
\foreign{\latntxt{çıq\=/maq}} \inlinegls{go.out-\Inf}.

\subsection*{Miscellaneous}

The symbol \zero{} represents zero elements,
i.e.\ segments that are not overtly realised, but that nonetheless contributes
to meaning.
Single\-/bar arrows express derivation;
\derivesto{} means `derives to' and \derivesfrom{} means `is derived from'.
Double\-/bar arrows express copying of items from other languages;
\copiedfrom{} means `is copied from'.
