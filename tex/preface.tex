\chapter{Preface}

This work is a reference specification of the Interturkic language (Latin: \latscript{Ortaqtürk tili}, Arabic: \arabscript{ارتاقتورک تیلی}).
The conventions described in this section will be used throughout this text.

Dashes delineate the boundaries between morphemes, both in example text and in glosses, e.g.\ \latscript{köz-ler-imiz-den} eye-\Pl{}-\Poss{}.\Fpl{}-\Abl{} `from my eyes'.
A dash to the left of an element marks it as a verbal stem, e.g.\ \latscript{kel-} `come'.
A dash to the right of an element marks it as a suffix, e.g.\ genitive case suffix \latscript{-nIñ}.

Parentheses around an individual phoneme denote that its inclusion is conditional on its environment.
If the enclosed phoneme is a vowel, then it surfaces only if the suffix is joined to a consonant final stem.
For example, the first-person possessive suffix \latscript{-(I)m} in \latscript{qalqan-ım} `my shield and \latscript{arpa-m} `my barley'.
On the other hand, if the enclosed phoneme is a consonant, then it surfaces only if the suffix is joined to a vowel final stem.
For example, the possessive dative case suffix \latscript{-(n)A} in \latscript{tawı-na} `to their (\Sg{}) mountain' and \latscript{közlerim-e} `to my eyes'.

Slashes enclose \emph{phonemic} transcriptions, while square brackets enclose \emph{phonetic} transcriptions.
For example, the phonemic transcription of \latscript{oquw} would be /ɔquw/, while the \emph{phonetic} transcription would be [ɔquː].
