# Interturkic langauge

This is a description of the grammar of Interturkic, a Turkic-based constructed language.

## Required packages

`bidi, caption, float, fontspec, gb4e, geometry, leipzig, longtable, multirow`

## Compilation instructions

```
xelatex interturkic-grammar.tex
makeglossaries interturkic-grammar.tex
xelatex interturkic-grammar.tex
```
